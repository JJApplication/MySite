Mgekssr文档

Mgekssr是一个可视化ssr爬虫网站

介绍

Mgekssr是一个基于Flask搭建的在线小飞机爬取网站

示例

实现原理

基于Flask1.0.3 ，pyquery ， requests

Dockerfile


FROM python:3.7
MAINTAINER landers liaorenj@gmail.com
WORKDIR /app
ADD . /app
RUN pip install -r requirements.txt
EXPOSE 80
RUN chmod +x /app/start.sh
ENTRYPOINT /app/start.sh

requirements.txt


gunicorn
flask
pyquery
requests
schedule

Docker镜像

$ docker pull landers1037/mgekflask

运行测试

$ docker run -itd -p 5000:80 mgekflask
